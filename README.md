## Installation

- Clone this repository -
  ```
  git clone https://github.com/inferiorak/RansoInferor
  ```

- Now go to cloned directory -
  ```
  !!! Note: Running the file will corrupt your current directory Files Only
  
  $ cd RansoInferor
  $ pip3 install -r requirements.txt
  $ clear
  $ python3 inferor.py 
  ```
